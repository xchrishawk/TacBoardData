Diagram

Trigger

Stage HUD TAD TGP MAV AIM-9
First Stage Enable PAC
Second Stage Fire Cannon

NOTE: PAC is disabled if AHCP GUN switch is set to GUNARM.

Weapon Release Button

State HUD TAD TGP MAV AIM-9
All Release Weapon

NOTE: Some weapons (including Paveway and JDAM) require button to be held for one second before release.

Master Mode Control Button (MMCB)

Press HUD TAD TGP MAV AIM-9
Short Toggle HUD
Long Select Air-to-Air Mode
