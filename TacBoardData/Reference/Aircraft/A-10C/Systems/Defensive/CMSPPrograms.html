CMSP Programs

Operation Modes

  • MAN: Pilot selects program and must manually start and stop dispensing countermeasures.
  • SEMI: CMSP chooses the best program for the detected threads. Pilot must manually start and stop dispensing.
  • AUTO: CMSP chooses the best program and automatically starts and stops dispensing.
