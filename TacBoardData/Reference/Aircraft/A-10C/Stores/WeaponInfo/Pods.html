Targeting

AN/AAQ-28 LITENING
Usage Targeting / Reconnaissance
Notes
  • Targeting pod with CCD and FLIR modes.
  • Provides target designation using laser and IR pointers.
  • Managed from the TGP page on the MFCD.

ECM

ALQ-131
Usage Electronic Countermeasures
Notes
  • Electronic countermeasures pod.
  • Used to jam radar signals to prevent acquisition of the aircraft.
  • Activated and deactivated using the CMS switch on the control stick.
ALQ-184
Usage Electronic Countermeasures
Notes
  • Updated and improved version of the ALQ-131 electronic countermeasures pod.
  • In DCS, the only difference is that the ALQ-184 is lighter. There is no performance difference.

Fuel Tanks

TK600
Usage Fuel Storage
Notes
  • External fuel tank containing an additional 600 gallons of fuel for long-distance rebasing missions.
  • A-10C can carry up to 3 external tanks on stations 4, 6, and 8.
  • Fuel is completely emptied from external tanks before internal fuel is used.
  • Tank has no self-sealing capability, and is never flown in combat.

Miscellaneous

AN/ASQ-T50 TCTS Pod
Usage Training / Flight Data Recording
Notes
  • Flight data recording pod used for training and evaluation purposes.
  • Has no function in DCS - cosmetic only.
BRU-42LS
Usage Equipment Rack
Notes
  • Listed under pods, however this is actually an empty triple ejector rack (TER).
  • Has no function in DCS - cosmetic only.
MXU-648 Travel Pod
Usage Cargo
Notes
  • Used to carry aircraft equipment such as intake covers and wheel chocks when rebasing.
  • Never carried in combat.
  • Has no function in DCS - cosmetic only.
Smokewinder
Usage Signaling / Aerobatics
Notes
  • Generator producing a smoke trail which follows the aircraft flight path.
